Lyra-JS

A tiny javascript library that makes code look neater, dates easier and much more. Lyra-JS came about while I was working as a freelancer, mainly moving projects from previous stacks to Node-JS. Different projects had dates in different formats, most of the time not in a valid format and since these projects were written in different programming languages there were functions not available in javascript. To solve these issues I wrote functions that I saw myself use repeatedly and that is how Lyra-JS came to be. Happy coding ðŸ™‚
